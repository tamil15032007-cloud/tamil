# Delicious Bites Restaurant Website

A simple restaurant-themed website built with HTML, CSS, and JavaScript.

## Features

- Home page with hero section
- Menu display
- About section
- Contact information
- Responsive design
- Smooth scrolling navigation

## Usage

Open `index.html` in your web browser to view the website.

## Customization

- Edit `index.html` to change content
- Modify `style.css` for styling
- Update `script.js` for interactivity

## Troubleshooting

If images don't load, replace placeholder URLs in `style.css` with actual image paths.

For local development, use a local server if needed for certain features.
